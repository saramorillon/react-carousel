# hooks

React hooks utilities

## Installation

With NPM

```bash
npm install @saramorillon/hooks
```

Or with Yarn

```bash
yarn add @saramorillon/hooks
```

## Usage

### useFetch

```jsx
import React from 'react'
import { useFetch } from '@saramorillon/hooks'

async function fetch() {
  // Fetch resource
}

function MyComp() {}
```
